Graph neural network-based FDD framework using a knowledge-based process topology graph.

- **Fault Detection:** Graph Auto-Encoder (GAE) + Mahalanobis Distance thresholding — unsupervised, FDR = 0.99, F1 = 0.97
- **Fault Diagnosis:** Attention-based Spatial Temporal GCN (ASTGCN) — 97.63% accuracy on 20 fault classes
- **Dataset:** Extended Tennessee Eastman Process (Harvard Dataverse)

**GitHub:** [TEP-Fault-Detection-Diagnosis-GAE-ASTGCN](https://github.com/Parth-B7999/TEP-Fault-Detection-Diagnosis-GAE-ASTGCN)  
**Paper:** [Digital Chemical Engineering, 2024](https://doi.org/10.1016/j.dche.2024.100158)
