Proposes a two-stage framework for fault detection and diagnosis (FDD) of multivariate process sensor data using knowledge-based graphs. A Graph Auto-Encoder (GAE) with Mahalanobis Distance thresholding detects faults unsupervised; an Attention-based Spatial Temporal GCN (ASTGCN) classifies the fault type. Demonstrated on the benchmark Tennessee Eastman Process achieving 97.63% accuracy across 20 fault classes.
